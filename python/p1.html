第1章 Pythonことはじめ
はじめに

みなさま Python はご存知でしょうか。というか知らなければこんなサイトには来ないと思います。^^;

Python は最近流行のスクリプト言語と呼ばれる言語で、GoogleやOSのLinuxでよく利用されています。日本でもじわじわと利用者が増えているという噂です。
またオブジェクト指向言語でもあり、クラスと呼ばれる概念を用いてプログラミングができます。

このサイトではプログラミングをある程度やってみたことがある人を対象に、Python とは何たるかをお伝えしていこうと思います。 もちろん、プログラミングをしたことが無い人でも、Python を使ったプログラミングがどのようなものかをイメージできるよう、ソースコードを詳しく解説しながら進めていきたいと考えています。 このサイトをみて Python に興味を持ち、自分でプログラミングしてみたいと感じたら、自らの手で開発環境の構築してみてくださいね。

あ、それとこのサイトで用いる Python のバージョンは 2.x 系 です。今後は 3.x 系が主流になりますが、筆者が 3.x 系がわからんのでこのまま進めます！orz

ごたくは置いておいて
