Python 講座

第1章 Pythonことはじめ
この章は Python のさわりの部分を説明しています。文法を手っ取り早く知りたい場合は第3章以降をご覧下さい。
はじめに

みなさま Python はご存知でしょうか。というか知らなければこんなサイトには来ないと思います。^^;

Python は最近流行のスクリプト言語と呼ばれる言語で、GoogleやOSのLinuxでよく利用されています。日本でもじわじわと利用者が増えているという噂です。
またオブジェクト指向言語でもあり、クラスと呼ばれる概念を用いてプログラミングができます。

このサイトではプログラミングをある程度やってみたことがある人を対象に、Python とは何たるかをお伝えしていこうと思います。 もちろん、プログラミングをしたことが無い人でも、Python を使ったプログラミングがどのようなものかをイメージできるよう、ソースコードを詳しく解説しながら進めていきたいと考えています。 このサイトをみて Python に興味を持ち、自分でプログラミングしてみたいと感じたら、自らの手で開発環境の構築してみてくださいね。

あ、それとこのサイトで用いる Python のバージョンは 2.x 系 です。今後は 3.x 系が主流になりますが、筆者が 3.x 系がわからんのでこのまま進めます！orz

ごたくは置いておいて

難しい話はさておき、Python を使うとこんな風にプログラムが書けるんだよ、というのを紹介していこうと思います。

まず始めに Hello world! です。なんというかプログラミングを始めると必ずこの Hello world! が付きまといますｗ ここでもそのしきたり（？）に倣い、素直に Hello world! の文字列を画面に出力してみたいと思います。

List 1. Hello world! in Python
print "Hello world!"

どうでしょう。これだけで画面に Hello world! と表示されます。

「で、これどうやって使うの？」

はい、使うためには Python 本体のプログラムとテキストエディタ（メモ帳とか）が必要です。また、Python のコードを書くための開発環境（IDEと呼ばれています）も用意されています。

次章では開発するための開発環境の紹介とその環境作り、実際に上のプログラムを動かしてみようと思います。
初めての人にはわからない単語がたくさんあるかもしれませんが、そんなことは無視して Python はこんなもんなんだという感覚をつかんでいきましょう。 意外にできちゃいますよ〜！
